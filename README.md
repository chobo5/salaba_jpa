## 개요
**사라바 RestAPI 프로젝트**
<br>
기존의 팀프로젝트 사라바를 전체 RestAPI로 재구성하였습니다. 

## 기술스택
DEV: `Java` `SpringBoot` `SpringSecurity``Jwt` `JPA` `QueryDsl` `Junit5`
<br>
DB: `MySql` `Redis`
<br>
문서화: `Swagger`
<br>
배포: `AWS` `Jenkins`
